# 全功能印刷企业知识库与培训系统开发计划

根据您的最新指示，系统将升级为**"知识管理 + 在线培训"**综合平台。除了知识库和事故案例，新增完整的培训考试体系。

## 1. 核心业务模块设计

### 1.1 知识管理模块 (Knowledge Base)
*   **通用文档**: 技术SOP、设备保养手册、原材料标准。
*   **事故案例库**: 结构化记录客户、金额、责任人、原因分析等。
*   **文件预览**: 支持 PDF, Word, Excel, 图片在线预览。

### 1.2 在线培训模块 (Training)
*   **培训课程**: 关联知识库文章或独立上传视频/PPT，支持"入职培训"、"岗位技能培训"等分类。
*   **学习记录**: 追踪员工对课程的学习进度 (已读/未读/学习时长)。

### 1.3 在线考试模块 (Exam)
*   **题库管理 (Question Bank)**:
    *   题型：单选、多选、判断。
    *   字段：题目内容、选项、正确答案、解析、关联知识点、难度。
*   **试卷管理 (Paper)**:
    *   组卷方式：手动选题或随机抽题。
    *   设置：考试时长、及格分数、补考次数限制。
*   **在线考试**:
    *   倒计时、答题卡、自动交卷。
*   **成绩管理**:
    *   自动阅卷 (客观题)。
    *   成绩查询与统计 (个人成绩单、部门合格率)。

## 2. 数据库模型扩展 (MongoDB)

新增以下模型：
*   **Question**: 题目模型。
*   **ExamPaper**: 试卷配置模型。
*   **ExamRecord**: 考试记录 (关联 User, ExamPaper, 答题详情, 分数, 是否及格)。
*   **Course**: 培训课程模型。
*   **StudyRecord**: 学习记录。

## 3. 权限与角色 (RBAC)

*   **超级管理员**: 全局权限。
*   **培训主管 (新增)**: 管理题库、试卷、课程，查看全员成绩。
*   **普通用户**: 学习课程、参加考试、查询个人成绩、管理个人知识文档。
