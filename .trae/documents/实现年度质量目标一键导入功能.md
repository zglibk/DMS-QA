我将为您实现“一键从往年导入年度目标”的功能，支持批量选择和复制，大幅提高创建效率。

### 1. 后端 API 扩展
在 `server/routes/quality-targets.js` 中新增复制接口：
- **路由**: `POST /copy`
- **功能**: 接收一组源目标 ID (`ids`) 和目标年份 (`targetYear`)，将这些记录复制为新一年的目标。
- **逻辑**:
  - 复制基础字段（目标名称、分类、计算公式、目标值、措施等）。
  - 更新关键字段：`TargetYear` 设为新年份，`CreatedBy/At` 更新为当前操作人/时间。
  - 状态重置为 `active` (进行中)。

### 2. 前端服务层更新
在 `frontend/src/services/qualityTargetsApi.js` 中添加：
- `copyQualityTargets(data)` 方法，对接后端复制接口。

### 3. 前端界面实现
在 `frontend/src/views/quality/targets/QualityTargets.vue` 中进行如下改造：
- **新增入口**: 在“导出数据”按钮旁增加“从往年导入”按钮。
- **新增弹窗 (`ImportDialog`)**:
  - **筛选区**: 选择“来源年份”（默认去年）和“目标年份”（默认今年）。
  - **预览区**: 展示来源年份的所有质量目标列表（复选框支持全选/多选）。
  - **操作区**: “确认导入”按钮。
- **交互流程**:
  1. 点击“从往年导入”。
  2. 弹窗自动加载去年的目标列表。
  3. 用户勾选需要沿用的目标（支持一键全选）。
  4. 点击确认，系统自动批量创建新一年的目标。
  5. 导入完成后刷新列表，用户可直接在列表上点击“编辑”对个别目标进行微调。

此方案既支持**批量复制**整个公司的年度目标体系，也支持**挑选复制**特定部门的目标，完美契合“一键复制并简单修改”的需求。
